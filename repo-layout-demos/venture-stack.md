> Thesis: build a portfolio of prediction products, monetize signal infrastructure, and continuously refresh edge through AI RnD.

| Tier | Repo | Role | Value Vector |
|---|---|---|---|
| Flagship | [mma-ai](https://github.com/bestisblessed/mma-ai) | Core product | Fight prediction engine and user-facing analysis |
| Flagship | [nfl-ai](https://github.com/bestisblessed/nfl-ai) | Core product | NFL data pipeline and matchup model output |
| Flagship | [mlb-ai](https://github.com/bestisblessed/mlb-ai) | Core product | MLB modeling and probabilistic decision support |
| Flagship | [mma-ai-swift-app](https://github.com/bestisblessed/mma-ai-swift-app) | Distribution | Mobile channel for product reach and retention |
| Utility | [odds-monitoring](https://github.com/bestisblessed/odds-monitoring) | Infrastructure | Market movement alerts and operational edge |
| Utility | [trading-bots-private](https://github.com/bestisblessed/trading-bots-private) | Execution | Private production strategy automation |
| Utility | [trading-bots-public](https://github.com/bestisblessed/trading-bots-public) | Validation | Public testbed for repeatable strategy logic |
| Utility | [oil-ai-dashboard](https://github.com/bestisblessed/oil-ai-dashboard) | Vertical product | Energy analytics with domain-specific monitoring |
| RnD | [ai-templates](https://github.com/bestisblessed/ai-templates) | Framework | Reusable patterns for AI agent capabilities |
| RnD | [ai-local-builds](https://github.com/bestisblessed/ai-local-builds) | Framework | Local model tooling and architecture testing |
| RnD | [deepseek](https://github.com/bestisblessed/deepseek) | Framework | Multi-provider model experimentation |
| RnD | [the-fight-predictor-agent](https://github.com/bestisblessed/the-fight-predictor-agent) | Experiment | Automated social distribution and agent behavior |

### Strategic Split
- Product Surface: `mma-ai`, `nfl-ai`, `mlb-ai`, `mma-ai-swift-app`
- Signal and Execution: `odds-monitoring`, `trading-bots-private`, `trading-bots-public`
- Innovation Engine: `ai-templates`, `ai-local-builds`, `deepseek`, `the-fight-predictor-agent`, `oil-ai-dashboard`
